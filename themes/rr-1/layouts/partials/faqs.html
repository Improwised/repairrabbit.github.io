Faqs

Why is it self hosted?

Because, We have seen that after a certain time your business need a custom development and that can't be possible with SAAS (software as a service). In case of RepairRabbit, you will get whole source code and you can customized as you need as per your business.

Can I get update on bug?

If that's a bug reported by all users, we will surely make it available to everyone at no cost.

How do you help me setup RepairRabbit?

We have pretty good installation guide. You can also send us an email for support.

Can you help me setup RepairRabbit for me?

Yes, we can help to setup RepairRabbit with one time minimum setup charge. But, we encourage you to see our installation guide and do it yourself without any cost.

What kind of server do you suggest me to host RepairRabbit?

We have tested RepairRabbit on shared hosting as well as cloud server. Please refer our hosting guide.

Do you help me customize RepairRabbit as per my business?

Yes, we are glad to customize RepairRabbit based on your business. It will be charged based on your feature request. Send us an email for the quote.

What does my licenses allow me to do with source code?

  • Install your source code on one registered domain.

  • Modify the source code and use it.

  • Allow all of your employee/associates to use RepairRabbit software for your business on registered domain.

What does my licenses not allow me to do with source code?

  • Sell or distribute any part of the original source code

  • Host source code on multiple domain

  • Host source code and sell service to other business
